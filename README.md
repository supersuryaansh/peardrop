## Peardrop

Share files privately over Hyperswarm.

Usage:

If I am online and seeding you can run it directly:

```
pear run pear://q63p3ban1yachhf17cn5tb5pkdc481c36xcm85ud87krj7a5q7qo
```

or git clone the directory and run yourself:

```
git clone https://github.com/supersuryaansh/peardrop.git
cd peardrop
npm i
pear run .
```
